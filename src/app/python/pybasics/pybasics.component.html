Python Basics

Python is a general purpose programming language. Its not intended for use in any particular domain or environment. Pyhton is a strongly typed language. Every object in the language generally has a type, at the same time python is dynamically typed (meaning there is no type checking of code prior to running it).

Python comes with a number of built-in datatypes. These include primitive scalar types like integers as well as collection types like dictionaries. These built-in types are powerful enough to be used alone for many programming needs, and they can be used as building blocks for creating more complex data types.
